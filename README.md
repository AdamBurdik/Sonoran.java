# Sonoran.java
Sonoran.java is a library that allows you to interact with the [Sonoran CAD](https://sonorancad.com/)

# How To Build
1. Clone the repository
```bash
   git clone https://github.com/adamBurdik/Sonoran.java
```
2. Navigate to the directory
```bash
   cd Sonoran.java
```
3. Build the module
```bash
   gradlew build
```
